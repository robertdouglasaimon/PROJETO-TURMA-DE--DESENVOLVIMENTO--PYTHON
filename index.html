SOBRE O CURSO

Durante esses intensivos 10 dias, exploramos conceitos fundamentais como números inteiros, números de ponto flutuante, strings, booleanos entre outros assuntos importantes. Através de uma combinação de teoria e prática, desenvolvemos uma compreensão desses conceitos essenciais, permitindo-nos avançar para projetos práticos.

Sob a orientação do Professor Roberto, criamos uma variedade de aplicativos, desde simples relógios digitais e calculadoras até jogos interativos como o famoso jogo da velha. Além disso, exploramos aplicações básicas de cadastro e aprendemos a criar funções para automatizar tarefas repetitivas.

Apesar do curto período de tempo, nossa turma composta por 18 alunos mergulhou em cada desafio, aproveitando ao máximo o momento de aprendizado e colaboração. O ritmo acelerado do curso nos desafiou a explorar o vasto potencial da linguagem Python.

Este curso foi uma experiência que nos equipou com habilidades valiosas para a introdução a tecnologia. Estamos gratos ao Professor e a Instituição por sua dedicação, expertise e paixão por compartilhar conhecimento.

SOBRE O PYTHON E A PROGRAMAÇÃO

Python se destaca como uma das linguagens de programação mais versáteis e poderosas, com aplicabilidade em uma ampla variedade de áreas da tecnologia. Independentemente do campo em que você atua - seja engenharia de software, análise de dados, desenvolvimento de jogos ou pesquisa acadêmica - dominar o Python pode abrir portas para oportunidades ilimitadas e impulsionar sua carreira para novos patamares de sucesso.

Números inteiros (int)

  • Representam números sem casas decimais, como 1, 2, -3, 100.
    São usados para contar, calcular e armazenar valores discretos.


    • Números de ponto flutuante (float)

  • Representam números com casas decimais, como 3.14, 5.50, -8.9.
    São usados para medidas, valores monetários e cálculos científicos.


    • Strings (str)

  • São sequências de caracteres, como letras, números, símbolos e espaços em branco.
    São usadas para armazenar textos, nomes, frases e mensagens.


    • Booleanos (bool)

  • Representam valores lógicos, True (verdadeiro) ou False (falso).
    São usados para tomar decisões, verificar condições e realizar comparações.

    Exercicios Práticos

    Durante o curso, exploramos uma ampla gama de atividades práticas projetadas para fortalecer nosso entendimento de conceitos fundamentais e desenvolver habilidades de programação em Python. Desde a criação de simples aplicativos de console até o desenvolvimento de jogos interativos, cada atividade nos desafiou a aplicar o que aprendemos de forma criativa e prática. Ao longo do processo,criamos projetos como construir relógios digitais, criar calculadoras funcionais, desenvolver o clássico jogo da velha e muito mais. Com uma abordagem prática e orientada a projetos, cada atividade nos proporcionou uma oportunidade de explorar o vasto potencial da linguagem Python e adquirir habilidades para o desenvolvimento de software.


    Tamagochi

    No projeto Tamagotchi, mergulhamos na criação de um simulador de animal de estimação virtual em Python. Exploramos o código-fonte deste icônico jogo e analisamos suas mecânicas e funcionalidades durante as aulas. Através dessa experiência, aprendemos sobre conceitos fundamentais de programação, como loops, condicionais e funções, enquanto desenvolvíamos nossa compreensão sobre estruturas de dados e lógica de programação. Este projeto proporcionou uma introdução prática e envolvente ao mundo da programação em Python, preparando-nos para aventuras mais desafiadoras e emocionantes ao longo do curso.

    tamagochi.py


    Jogo Da Velha

    No projeto do Jogo da Velha desenvolvido em Pygame, embarcamos em uma jornada para criar uma versão digital deste clássico jogo de tabuleiro. Utilizando a biblioteca Pygame, exploramos os fundamentos da programação de jogos e gráficos em Python, enquanto desenvolvemos a construção de interfaces interativas e dinâmicas. Durante as aulas, analisamos o código-fonte do jogo, aprendendo sobre detecção de eventos, manipulação de sprites e implementação de lógica de jogo. Este projeto nos mostrou o potencial criativo e versátil da linguagem Python no desenvolvimento de jogos.

    velha.py


    Relógio Digital

    No projeto do Relógio Digital desenvolvido em Tkinter, adentramos no mundo da interface gráfica de usuário (GUI) em Python. Utilizando a biblioteca Tkinter, exploramos a construção de uma aplicação que exibe a hora atual de forma dinâmica em uma janela gráfica. Durante o desenvolvimento, aprendemos sobre widgets, layouts e eventos em Tkinter, permitindo-nos criar uma interface interativa e amigável para o usuário. Este projeto nos proporcionou uma introdução prática à criação de GUIs em Python, destacando a simplicidade e eficiência do Tkinter para construir aplicações desktop intuitivas.

    relogio.py


    Cadastro de clientes

    No projeto do aplicativo de cadastro de clientes desenvolvido com PyQt5.QtWidgets, mergulhamos na criação de interfaces gráficas modernas e interativas em Python. Utilizando a biblioteca PyQt5.QtWidgets, construimos uma aplicação que permite o cadastro e gerenciamento de clientes de forma intuitiva e eficiente. Durante o desenvolvimento, aprendemos sobre a criação de janelas, widgets personalizados, sinais e slots, proporcionando uma experiência de usuário fluida e agradável. Este projeto nos permitiu explorar o poder e a flexibilidade do PyQt5.QtWidgets para desenvolver aplicações desktop robustas e profissionais em Python.

    Cadastro de clientes.py

    Bibliografia

    Explore o vasto universo de bibliotecas que impulsionam o desenvolvimento em Python! Das ferramentas essenciais para criação de interfaces gráficas e jogos até as poderosas bibliotecas para visualização de dados e construção de aplicativos web, cada uma dessas bibliotecas oferece recursos e funcionalidades únicas para expandir os horizontes de programação.


    Tkinter

    Tkinter é uma biblioteca de interface gráfica (GUI) padrão do Python, que fornece uma maneira simples e eficaz de criar interfaces de usuário para aplicativos desktop. Com o Tkinter, os desenvolvedores podem criar janelas, botões, caixas de texto, menus e outros elementos de interface de forma rápida e fácil. Ele é amplamente utilizado devido à sua facilidade de uso e à sua capacidade de criar aplicativos com uma aparência nativa em diferentes sistemas operacionais, incluindo Windows, macOS e Linux.

    https://docs.python.org/3/library/tkinter.html


    Pygame

    Pygame é uma biblioteca de desenvolvimento de jogos em Python que facilita a criação de jogos 2D interativos. Ela oferece uma ampla gama de funcionalidades para manipulação de gráficos, som, entrada de usuário e física, permitindo aos desenvolvedores criar jogos envolventes e dinâmicos. Com o Pygame, é possível criar desde simples jogos até projetos mais complexos, aproveitando a simplicidade e flexibilidade oferecidas pela linguagem Python. É uma escolha popular para iniciantes e desenvolvedores experientes devido à sua facilidade de aprendizado e à sua capacidade de criar jogos de alta qualidade.

    https://www.pygame.org/news


    Matplotlib

    Matplotlib é uma biblioteca de visualização de dados em Python amplamente utilizada para criar gráficos e plots de alta qualidade. Com o Matplotlib, os desenvolvedores podem gerar uma variedade de gráficos, incluindo gráficos de linha, dispersão, barras, histogramas e muito mais. Ele oferece controle detalhado sobre cada aspecto do gráfico, permitindo personalização completa da aparência e estilo. Além disso, o Matplotlib é integrado perfeitamente com outras bibliotecas de análise de dados, como NumPy e Pandas, tornando-o uma escolha popular para análise e visualização de dados em Python.

    https://matplotlib.org


    Flask

    Flask é um framework leve e flexível em Python para construir aplicativos web. Ele oferece uma abordagem simples e direta para o desenvolvimento de aplicativos web, permitindo que os desenvolvedores criem rapidamente aplicativos web poderosos e escaláveis. Com o Flask, é fácil criar rotas, manipular solicitações HTTP, renderizar modelos HTML e muito mais. Ele também oferece extensões adicionais para funcionalidades avançadas, como autenticação de usuário, integração de banco de dados e suporte a APIs RESTful. Flask é uma escolha popular para criar aplicativos web desde projetos simples até grandes sistemas de produção, devido à sua simplicidade e flexibilidade.

    https://flask.palletsprojects.com/en/3.0.x/
